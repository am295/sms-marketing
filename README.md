# 📱 2025년 12월 SMS 마케팅 전략 제안서

> 크리스마스와 연말 시즌, 고객과의 직접적인 소통을 통해 매출 극대화와 브랜드 충성도를 높이는 전략적 접근

## 📋 프로젝트 개요

2025년 12월 시즌을 겨냥한 종합 SMS 마케팅 전략 제안서입니다. 
크리스마스, 연말 특가, 고객 감사 캠페인 등 시즌별 맞춤 전략과 실제 SMS 예시를 포함하고 있습니다.

### 🎯 프로젝트 정보
- **기간**: 2025년 12월
- **작성일**: 2025년 11월
- **타겟**: 모든 업종 적용 가능
- **포맷**: 인터랙티브 HTML 웹 문서

## ✨ 주요 특징

### 📊 포함된 콘텐츠
1. **Executive Summary** - 12월 SMS 마케팅의 중요성
2. **전문가 인사이트** - 데이터 기반 핵심 전략
3. **Market Insight** - 업계 평균 성과 지표
4. **Target Segmentation** - 업종별 타겟 전략
5. **SMS Strategy** - 3대 핵심 전략 프레임워크
6. **Campaign Type 1** - 크리스마스 캠페인
7. **Campaign Type 2** - 연말 특가 캠페인
8. **Campaign Type 3** - 고객 감사 캠페인
9. **Campaign Timeline** - 12월 주차별 발송 전략
10. **Expected Results** - 기대 효과 및 KPI

### 🎨 인터랙티브 기능
- ✅ **반응형 디자인** - 모든 디바이스 최적화
- ✅ **이미지 확대 기능** - SMS 예시 클릭 시 전체화면 보기
- ✅ **부드러운 애니메이션** - 페이드인, 줌인 효과
- ✅ **프린트 최적화** - PDF 변환 시 깔끔한 레이아웃
- ✅ **현대적인 UI/UX** - 그라디언트, 카드 디자인

## 🚀 사용 방법

### 1. 로컬에서 실행

```bash
# 저장소 클론
git clone https://github.com/yourusername/sms-marketing-proposal-2025.git

# 디렉토리 이동
cd sms-marketing-proposal-2025

# index.html 파일을 브라우저로 열기
open index.html
# 또는
start index.html
```

### 2. GitHub Pages로 배포

1. GitHub 저장소 Settings로 이동
2. Pages 섹션 선택
3. Source를 `main` 브랜치로 설정
4. 자동 생성된 URL로 접속

### 3. 직접 파일 열기

- `index.html` - 미리보기 페이지
- `sms_marketing_proposal.html` - 실제 제안서 문서
